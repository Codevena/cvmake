# Roadmap

This is a living document. PRs and issues are welcome on any item below.

## v0.2 — Editor polish
- Responsive editor layout (mobile + tablet)
- Locale switcher in the editor form (live UI; today the demo switch requires a reload)
- Drag & drop photo upload
- Undo/redo
- Inline rename for skill categories

## v0.3 — Templates & content
- Cover-letter templates
- More template variants (community contributions welcome — see `.github/ISSUE_TEMPLATE/template_idea.yml`)
- ATS-friendly template variants

## v0.4 — Quality of life
- Self-hostable one-click installer
- Cloud sync (opt-in)
- More palette variations per template

## Done
- 12 launch templates (academic, bauhaus, classic-serif, corporate, creative-accent, editorial, magazine, modern-minimal, monochrome-dark, noir, swiss, tech-dev)
- CLI + web editor
- Multilingual support (English + German)
- MIT license, OSS publishing pipeline
